# Chapter 2: Unspoken Words

As time goes on, you can simply add a new file to the `src/content/chapters` folder. 

Name it something like `03-next-chapter.md` and the app will automatically pick it up, parse the title, and add it to the reading list.

* The chapters are sorted automatically based on their filenames.
* You don't have to touch the code anymore—just write your heart out in these Markdown files.
